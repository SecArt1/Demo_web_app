NICE Framework Mapping

We have mapped the WC-CSE to the following NICE Categories:

Application and Analysis (Applying, Analyzing)

Cyber Defense Analyst (PR-CDA-001), Incident Responder (SP-IR-001), Vulnerability Assessment Analyst (PR-VAM-001)

Strategic Development (Evaluating, Creating)

Security Architect (OV-ARC-001), Cybersecurity Manager/Administrator (OV-MGT-001), Systems Security Analyst (PR-INF-001)

Specialized Technical Skills

Secure Software Developer (SP-DEV-001), Network Operations Specialist (OM-NET-001), Technical Support Specialist (SP-SPT-001)

Collaboration, Agile Development, Professional Skills

Cyber Instruction (TR-CIN-001), Project Manager (PM-MGT-001), Product Support Manager (SP-PMM-001)

Ethics and Communication

All Roles: Ethical behavior and effective communication are fundamental across all cybersecurity roles. These skills ensure that professionals can work within legal and ethical boundaries and communicate security concepts effectively.

Advanced Technical Skills

Penetration Tester (SP-PEN-001), Forensics Analyst (IN-FOR-001), Security Operations Center (SOC) Analyst (SP-ANL-001)
